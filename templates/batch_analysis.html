Batch Processing Tips
Requirements
  • Each image should contain all four test areas (Anti-A, Anti-B, Anti-D, H Antigen)
  • Images should be clear and well-lit
  • Supported formats: JPG, JPEG, PNG
  • Maximum file size: 5MB per image
Optimization
  • Process up to 20 images at once for optimal performance
  • Use consistent lighting conditions for all samples
  • Standardized image format yields more consistent results
  • Wait for processing to complete before uploading more batches
Processing time depends on the number and size of images. Larger batches may take several minutes to complete.
